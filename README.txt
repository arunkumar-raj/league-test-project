== Description ==

Create a custom WordPress plugin that will make it possible to store football/sports teams categorized by the league they are from and list them with a custom-made Elementor widget.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `ltp.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress and make sure the Elementor plugin is active
1. Edit pages with Elementor and search for teams or league or sports or football in the Elementor Widget Search bar and use it on the page.
